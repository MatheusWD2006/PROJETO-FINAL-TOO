Meu trabalho simula a questão das culturas agrículas e suas épocas do ano. Ele é composto de 5 classes principais: Cultura, classe abstrata que traz objetos base de uma cultura; Estacao, que traz as estações do ano; CulturaEstacao, que herda de cultura e se associa com uma estação já existente; CulturaAnoTodo, herda de cultura e simula o plantio no ano todo e statusCultura que se associa com uma CulturaEstacao e simula o plantio dessa cultura. Também tem 2 arquivos de padrões de projeto, um para criação de objetos CulturaAnoTodo e CulturaEstacao, onde escondemos a lógica por trás deles e só instanciamos os objetos, e o outro é uma padrão Strategy para cria um aviso para o objeto aviso presente na classe Estacao. Esse aviso é gerado a partir do nome da estação.
